Aadhaar Redaction API

The Aadhaar Redaction API is designed to automatically detect and obscure Aadhaar numbers (also known as the Unique Identification Number (UID) ) within documents and images.

The API leverages pattern recognition techniques to identify the last 8 digits of the Aadhaar number within a given document or image.

The masked Aadhaar number maintains the structural integrity of the original document while ensuring that the sensitive information remains hidden.
